# HeadsetControl-Gui
## HeadsetControl Gui is an Gui app that allows you to manage your logitech headset on linux 
### this app is build using lovable

### Requirements 
- https://github.com/Sapd/HeadsetControl
- Golang

## How to install 
- first install HeadsetControl
- then run ` go build `
- then run `./main`

## why i am building this
i think logitech is building great headsets but i hate it that they dont build software to manage it under linux even for the expensive headsets that costs like 200€
